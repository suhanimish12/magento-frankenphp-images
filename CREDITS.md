## Source Project

This repository contains Docker image definitions extracted from [magento-docker-frankenphp](https://github.com/mohaelmrabet/magento-docker-frankenphp), a complete Magento 2 development environment using FrankenPHP.

## Technologies

This project is built with and depends on these amazing technologies:

- **[FrankenPHP](https://frankenphp.dev/)** - The modern PHP app server, built on top of Caddy
- **[Caddy](https://caddyserver.com/)** - Fast and extensible multi-platform HTTP/1-2-3 web server with automatic HTTPS
- **[PHP](https://www.php.net/)** - The PHP programming language
- **[Composer](https://getcomposer.org/)** - Dependency Manager for PHP
- **[Magento 2](https://business.adobe.com/products/magento/magento-commerce.html)** - Open-source e-commerce platform
- **[Docker](https://www.docker.com/)** - Container platform
